# EXPERIMENT 04 – 8-BIT SHIFT REGISTER USING FLIP-FLOPS

## AIM
To design and simulate an 8-bit shift register using D flip-flops.

## THEORY

A **shift register** is a sequential circuit used to store and shift binary data.

### D Flip-Flop

A D flip-flop stores one bit of data.

**Characteristic Equation:**


Q(next) = D


### Working of Shift Register

- Data shifts from one flip-flop to the next on each clock pulse.
- All flip-flops are connected in series.

### Types of Shift Registers

- **SISO (Serial In Serial Out)**
- **SIPO (Serial In Parallel Out)**
- **PISO (Parallel In Serial Out)**
- **PIPO (Parallel In Parallel Out)**

### Applications

- Data storage
- Data transfer
- Serial communication

## PROCEDURE

1. Connect 8 D flip-flops in series.
2. Provide a common clock signal.
3. Apply serial input.
4. Trigger clock pulses.
5. Observe shifting of data.

## RESULT

The 8-bit shift register was successfully implemented.
