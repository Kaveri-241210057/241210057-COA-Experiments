# EXPERIMENT 06 – SHIFT MICRO-OPERATIONS

## AIM
To implement logical, arithmetic, and rotate shift operations.

## THEORY

Shift operations move bits left or right within a register.

### Types of Shift Operations

- **Logical Shift**  
  Vacated positions are filled with 0.

- **Arithmetic Shift**  
  Preserves sign bit.

- **Rotate Shift**  
  Bits are rotated within the register.

### Shift Operations

**Left Shift**


A << 1 = A × 2


**Right Shift**


A >> 1 = A / 2


### Applications

- Multiplication and division
- Bit manipulation
- Data processing

## PROCEDURE

1. Write a C program for shift operations.
2. Compile and execute.
3. Observe output changes.

## RESULT

All shift micro-operations were successfully implemented and verified.
