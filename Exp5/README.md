# EXPERIMENT 05 – ADDRESSING MODES USING GDB

## AIM
To study different addressing modes using C and GDB.

## THEORY

Addressing modes define how operands are accessed in memory.

### Types of Addressing Modes

- **Immediate Addressing**  
  Operand is directly specified.

- **Direct Addressing**  
  Address of operand is directly accessed.

- **Register Addressing**  
  Data is stored in CPU registers.

- **Indirect Addressing**  
  Address is stored in a pointer.

### Pointers

- Store memory addresses
- Used for indirect addressing

### Importance

- Efficient memory access
- Enables dynamic data handling

## PROCEDURE

1. Write a program using variables and pointers.
2. Compile with `-g`.
3. Run using GDB.
4. Use commands:

- `print a`
- `print &a`
- `print p`
- `print *p`

## RESULT

All addressing modes were successfully studied and verified.
